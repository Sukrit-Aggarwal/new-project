Step 1: Scope Creation

Description

AUD scope is a BitBucket repository belonging to a single team and which contain all AUD definition (Component and Dependencies) for this team.

Two types of scope exist:

Infrastructure scope
- that are under infrastructure team's responsibility
Application scope
- corresponds to a functional split and contains one APP

Infrastucture and application scopes data should not be merged in the same scope as they have different SLA/lifecycle and ownership/access rights.
