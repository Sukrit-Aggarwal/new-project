Step 4: Dependency Instances

Description

Dependency instance allows you to specify for a given instance of your Component which real Component instance you rely on.

The dependency instance is the way to declare an instance of a connectivity (between 2 component_instances).

This object has to reference the targeted_service of the destination component to know port and protocol to use.

Before, each connectivity between the same component_instance had to be declared for each service used.
